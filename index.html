gitの使い方～初めてのgithub～

Githubとはファイルのバージョンを管理してくれるもの

やることコマンド
ユーザー設定をする。git config --global user.name "ユーザ名"
git config --global user.email メールアドレス
gitの初期化をする。gitの管理下に置きたいファイルで実行する。git init
コミット待ちの状態にする。.のあとに何も書かないと全部のファイルをやるgit add.
コミットする。git commit -m "コメント"
gitのリポジトリに登録する。-uは最初だけ入れるgit push (-u) origin master
