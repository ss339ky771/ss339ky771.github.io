gitの使い方～初めてのgithub～

Githubとはファイルのバージョンを管理してくれるもの

ファイルのあげ方

やることコマンド
ユーザー設定をする。初めて使う時だけやる。git config --global user.name "ユーザ名"
git config --global user.email "メールアドレス"
gitの初期化をする。gitの管理下に置きたいファイルで実行する。.gitのファイルが出来ているかかくにんをする。git init
コミット待ちの状態にする。.のあとに何も書かないと全部のファイルをやるgit add.
コミットする。git commit -m "コメント"
gitのリポジトリに登録する。-uは最初だけ入れる。originのあとは枝の名前git push (-u) origin master

ほかのコマンドメモ

説明コマンド
susiという枝を作るgit branch susi
susiという枝に移動git checkout susi
gitの状態を見るgit status
コミットのログを見るgit log
